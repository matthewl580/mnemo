# Code of Conduct

Mnemo is a passion project and an open-source community. The goal is to make collaboration productive, respectful, and sustainable.

## Expected behavior

Participants are expected to:

- be respectful and constructive
- assume good intent when reasonable
- give feedback about work, not personal attacks
- respect maintainer time and boundaries
- keep discussions relevant to the project
- accept decisions once maintainers have made them

## Unacceptable behavior

The following behavior is not acceptable:

- harassment, threats, or intimidation
- hateful, discriminatory, or sexually explicit language
- personal attacks or repeated hostile behavior
- spam or low-effort disruption
- doxxing or publishing private information
- pressuring maintainers or contributors for unpaid work
- repeatedly ignoring project rules or maintainer decisions

## Enforcement

The project owner and maintainers may moderate issues, discussions, pull requests, and community spaces.

Depending on the situation, maintainers may:

- edit or remove comments
- lock discussions
- close issues or pull requests
- warn participants
- temporarily or permanently block participants from official project spaces

Maintainer decisions are final for official Mnemo project spaces.
